Demo 1 - Fixed column width

This forces all elements into a single column width

Width: px

Demo 2 - use existing widths

This does not change the widths of the elements, and tries to determine columns sizes autmatically. In general this will work better if the variety of actual widths are few and are multiples of eachother, e.g. 50, 150, 300 pixels.

Demo 3 - defining a set of possible widths

This will force the element widths to conform to the supplied set of widths. The columns are then calculated automatically. As with demo 2, this works best when the possible width set is small and are multiples of eachother.

Allowable widths: (pixels)
